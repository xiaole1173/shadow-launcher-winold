# Icons

启动器图标资源目录。

## 命名规范

```
<category>_<name>_<size>.svg
```

示例：
```
btn_download_24.svg        # 下载按钮 24x24
btn_pause_24.svg           # 暂停按钮 24x24
status_checkmark_16.svg    # 状态勾号 16x16
nav_home_32.svg            # 导航首页 32x32
```

## 规范

- 格式: SVG（矢量），可选 PNG 备用
- 按钮类: 24x24, 32x32
- 状态指示: 12x12, 16x16
- 导航图标: 32x32
- 颜色: 使用 currentColor 或启动器主题色系 (#5d6fe0, #4bc870, #c06050)
